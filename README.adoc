== Installation

Clone repository and launch `install_debian.sh`

== Uninstallation

- Remove the file `~/.wsl-starter`
- Edit your `~/.bashrc` and remove the last lines :
----
if [ -f ~/.wsl-starter ]; then
  . ~/.wsl-starter
fi`
----
- (optionnal) you can uninstall zsh, docker if you want
